Use GitHub and Hexo to build a Blog

  • Website

This website is built by Hexo + Github(yilia) + Godaddy + Dnspod.

You should have already installed NodeJS, which supports the node.js based framework, Hexo.

Install Hexo

npm install -g hexo-cli

Initialize a hexo website

hexo init Blog
