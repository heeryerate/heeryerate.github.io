Handy Commands


screen: screen command is very helpful if you want to have multiple shells running in one terminal window. Also, it can help you run a program persistently (even if a terminal window breaks or closes).

  1. Create a session. You can easily exit current session by Ctrl -a d

  2. List all sessions you have

  3. Reattach a session

tee: tee command is used to store and view (both at the same time) the output of any other command.

  1. Write output to stdout, and also to a file (or multiple files)

  2. By default tee command overwrites the file, you can use -a option to append to the file
