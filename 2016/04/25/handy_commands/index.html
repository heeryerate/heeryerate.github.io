Handy commands

  • Linux
tech6

screen: screen command is very helpful if you want to have multiple shells running in one terminal window. Also, it can help you run a program persistently (even if a terminal window breaks or closes).

  1. Create a session. You can easily exit current session by Ctrl -a d or terminate a session by Ctrl -r d

  2. List all sessions you have

  3. Reattach a session

tee: tee command is used to store and view (both at the same time) the output of any other command.

  1. Write output to stdout, and also to a file (or multiple files)

  2. By default tee command overwrites the file, you can use -a option to append to the file

tldr: tldr means too long, donâ€™t read :). It is a collection of simplified and community-drive man pages. see tldr. A web client is available as well.

awk: awk is a text pattern scanning and processing language.

  1. Print column of field of a file ($0$ stands for the whole line and $n$ is the string at nth column.)

  2. One could specific field separator by the option -F

  3. Print column by pattern (e.g. /a/ matches the line including the letter a)

  4. Replace unwanted character (e.g. replace chacter a to b in forth column. b could be whitespace, therefore delete)
