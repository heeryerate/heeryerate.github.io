Using mpi4py

  • python
programming1

MPI stands for Message passing interface and is used to write parallel programs in a distributed system such as a Linux cluster with distributed memory. Usually, MPI is implemented by MPICH or OpenMPI and works in C. During the last two years I’m working on some projects using MPI in C++, even there exist great library like boost which provides an easy wrapper of MPI, what I have tried recently in python is incredibly friendly to use.

This short blog is to demonstrate the use of MPI in python with a module named mpi4py. It is very easy and straightforward to use.

First, I will show how to use mpi4py and later on will show how to install and use mpi4py on both linux and windows.
