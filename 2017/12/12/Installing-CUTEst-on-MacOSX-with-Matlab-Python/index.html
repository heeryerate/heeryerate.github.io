Installing CUTEst on MacOSX with Matlab

  • matlab
optimization4

CUTEst is the latest evolution of CUTE, the constrained and unconstrained testing environment for numerical optimization. Detailed instruction can be found at CUTEst wiki, CUTEst paper and CUSTst paper. Currently, you could have 1285 problems and 171 unconstrained problems, with function evaluation (objective value, gradient, hessian) available.

In this blog, I’ll show how to create the Matlab interface for CUTEst. I’m installing on macOS High Sierra, Version 10.13.2, Matlab 2017b and Xcode 9.3, it should be similar to other setting as well.
